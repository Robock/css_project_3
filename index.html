What is HTML?

HTML is short for Hypertext Markup Language. It is a file system by which individuals communicate with computers. We write code in HTML format. Just like we write text in Microsoft Word to communicate with other humans, we use Text Editors to communicate with computers. A Text Editor is a piece of software that is used to write, edit and save code. We have a few options of text editors that we can use like Atom, Sublime Text, Brackets, etc.

In order to write code, we need to think like a programmer. The 5 ways programmers think are:

Procedural Thinking
A method of thinking that involves several steps.
Abstract Thinking
Finding similarities in different things to create Generality.
Systematic Debugging
Finding possible causes of problems and fixing it.
Technological Empathy
A computer is a tool that outputs information based on what a user inputs and programmers need to be mindful of that.
Systems Thinking
Breaking big problems into smaller pieces.

The World Wide Web

The major components of the web are: 1) Browser, 2) Internet, 3) HTTP and 4) Servers. A browser is something like an internet explorer, chrome, firefox or safari that makes request via the internet to servers. These requests are made using an internet connection and a protocol called HTTP. HTTP stands for Hypertext Transfer Protocol. Servers are dedicated computers that respond to the request from users with files that is displayed in the users' browser.

Make a stylish web page

Both HTML and CSS are languages. HTML controls the "structure" while CSS controls "style". The analogy of a house's structure being similar to HTML while the paint and decoration of the house is similar to what CSS does to web pages is very helpful to understand web page design. Along the same vein, you can think of JavaScript as an interactive tool in the house like a tv remote controller or garage door opener. Document Object Model (DOM) is a tree-like structure of a page. This structure can easily be seen by using Developer Tools or Dev Tools in browsers. Another little caveat I learned about web pages is that you can think of everything on a web page as boxes and these boxes useDiv tag to differentiate themselves from other boxes. Class attributes are used as labels for these boxes. Using Div tags helps group similar things together in a box thereby giving our code a structure. This practice also helps to modify the code in the future because related content is grouped together.

What is CSS?

CSS stands for Cascading Style Sheets. In a cascading relationship, the most specific rule at the bottom of the cascading rules applies to everything. A rule in CSS is made in a form of a declaration. A declaration is enclosed in a pair of curly braces having a property followed by a colon and a value followed by a semicolon. Each declaration uses selectors. A selector is a CSS code that selects a specific style for a component of an HTML element. Selector is basically the name you use in order to target the elements that are assigned to a class or id attribute in HTML. A class refers to a group of elements that can be styled together while ID's are unique identifiers that uniquely identify an element in HTML.

The 3 ways to include CSS styling in web pages are:

  1. Write CSS in the head of the HTML file.
  2. Link your HTML to a separate CSS file.
  3. Write your style inline with your HTML (not recommonded).
